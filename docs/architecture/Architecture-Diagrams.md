### What Production Architecture Uses
| Layer | Services | Purpose |
|-------|----------|---------|
| **Edge** | CloudFront, WAF, Cognito | CDN, security, authentication |
| **App** | Amplify, API Gateway, App Runner | Frontend + Backend hosting |
| **AI** | Bedrock (Claude 3.5), Textract | LLM analysis + PDF processing |
| **Data** | Aurora, OpenSearch, S3 | Structured data + vectors + files |
| **Ops** | CloudWatch, Secrets Manager, CloudTrail | Monitoring + security + compliance |
| **Total** | **12 AWS services** | **Est. $750-1,200/month** |
